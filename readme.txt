RB Site Social Links plugin use for your site social links add.

== Description ==

RB Site Social Links plugin use for dynamically add your website social icon link. You need change only your social account link.

== Installation ==

### From your WordPress dashboard

= Visit =
Plugins > Add New

= Search =
for "RB Site Social Links"

= Install and Activate =
"RB Site Social Links" from your Plugins page

= Go to =
Setting -> General Page

= Links add =
Add your website's social media links
